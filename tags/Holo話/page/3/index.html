敨放 (tháu-pàng) 佇前

（更新日：2015-04-14）

對拄別細漢的[--ê]彼冬
新穎[ínn]煞[suah]入去白色的制限
色水愈濟　出賣愈深
lè-suh 軟幼　相縛絚絚[ân-ân]
胸坎的意志
束縛和[hām]視線千萬　無形帶殘忍
比並中　揀掉家己的自由

少年家趒[tiô]起來投籃　無牽無掛
阮卻佇無光的綿柔監獄
只有所哭的汗水　湠[tuànn]做紅紅的疹
佇間仔內用癢疼控訴
邊仔壓迫的繏巡[sǹg-sûn]

敢講怹[in]的軀體較袂唌[siânn]阮[--gún]？
只是虛權嵌牢自覺
嘛踏阮　夯輕輕的重枷

彼工青籠籠[tshenn lang-lang]的冊
閣逼阮扛一橫無平的時
對遠遠个寒冷　不滿若火煏[piak]出
趁海風燒甲[kah]這岸
脫離跤帛[kha-pe̍h]　毋通再並[phīng]
靠勇氣足足，飛佇青空中
捋[lua̍h]陽春个燒暖

就算面對無知的輕薄[khing-po̍k]
佮漚朽[àu-hiú]的框殼[khing-khak]

我 ê 故鄉－－北溝 (Pak-kau)

（最後更新佇：2014-08-21）

註：本文是台灣閩南語（Holo 話）書寫練習，拼音是台羅拼音，用字盡量用教育部常用詞辭典，看無的[--ê]通去彼个站揣[tshuē]。本文內底使用 <ruby>，用 Firefox 的建議先裝 HTML Ruby 來看較順序[khah sūn-sī]。

逐个[ê]人攏有伊的[--ê]故鄉，毋[m̄]管是出世地、成長地，抑[iah]毋管 蹛[tuà] 偌[guā]久，捷捷[tsia̍p-tsia̍p]會刻入 in[其之] 心內，成做人格的重要部份。足濟[tsiok tsē]綴[tuè]老蔣政府來台灣的文人，比論琦君[Kî-kun]，袂當[buē tàng]無記故鄉的風、土、水、情，寫出袂少的懷鄉文學；黃昏的故鄉、彼時的形影，嘛不時咧[teh]叫著予[hōo]當局貼[tah]「烏名單」的智識[tì-sik]分子。1在水[tsāi-tsúi]我無像 in[其等] 按呢[án-ne]強強無路轉故鄉，總是[tsóng--sī]離鄉讀冊的彼段日子，有時會真數[siàu]念。
