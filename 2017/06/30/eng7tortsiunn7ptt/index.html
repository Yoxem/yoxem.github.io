жң¬ж–ҮеҸғиҖғ How to use SSH with Tor? - Ask Ubuntu

иӢҘжҳҜжұқ[lГӯ]ж¬І[beh]и—Ҹ[tshГ ng]е®¶е·ұзҡ„[--ГӘ] IP дҪҮ[tД«] ptt йҖҷдёӘ[tsit-ГӘ] BBSпјҢйҖҡдҪҝз”Ё torгҖӮдҪҮ Linux зҡ„иҷӣзөӮз«Ҝж©ҹпјҢйҖҡз”Ёд»ҘдёӢжҢҮд»Өпјҡ

д»Је…ҲпјҢж„ӣ[Г i]зўәиӘҚе®¶е·ұ[ka-kД«]жңүиЈқ torify з„ЎпјҹиӢҘжҳҜдҪҮ Linux Mint 18пјҢйҖҡз”Ёпјҡ

apt install tor

иЈқгҖӮ

ж¬Ій–Ӣ ptt зҡ„жҷӮпјҢж„ӣе…ҲжӢҚй–Ӣ[phah-khui]иҷӣзөӮз«Ҝж©ҹпјҢй–Ӣж–°зҡ„еҲҶй ҒпјҢијёе…ҘtorйҖЈз”І[kah] Tor з¶Іи·ҜгҖӮ

й ҶеҲ©йҖЈжҺҘеҫҢпјҢй–Ј[koh]й–Ӣж–°еҲҶй ҒпјҢијёе…ҘжҢҮд»Ө

torify ssh bbsu@ptt.cc

е°ұйҖҡйҖЈеҲ° ptt.cc зҹЈпјҢжҜӢйҒҺ[mМ…-koh]йҖҹеәҰијғ[khah]ж…ўиҖҢдё”йҲҚйҲҚ[tЕ«n-tЕ«n]гҖӮ

иҖҢдё”з”Ё tor й–Ӣж–°зҡ„еҸЈеә§ (account) д»ҘеҫҢпјҢдёҖе®ҡж„ӣй–Ӣ tor жүҚ[tsiah]йҖҡ[thang]зҷ»е…ҘйҖҷдёӘеҸЈеә§гҖӮд»Ҙе…Қ IP дәҲдәә[hЕ‘ng]зҷјзҸҫгҖӮ
